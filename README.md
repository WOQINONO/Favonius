# Favonius
智能题库管理器：轻松创建自己的智能题库，自动配置每次练习，练习内容智能编排，支持AI大模型接入自动批阅

在线帮助文档：[查看](https://www.kdocs.cn/l/ctBY6IcVpsUH)

更新记录：[查看](https://www.kdocs.cn/l/cm8NoUhbYTlC)

此程序在 Windows 11 家庭中文版（25H2, x64平台）、Windows 11 Consumer Edition（x64平台）测试正常运行。
本应用程序的开发使用了其他开源项目、字体和图标，你可以在LICENSES文件夹中，或者软件“查看第三方模块版权声明”中找到相关说明。

本程序是在Windows平台上运行的自创建自练习题库框架软件。用户可自主创建专属题库，如同一个自己的笔记本，也能直接导入现成文本快速生成题库；用户除了可以手动指定练习内容外，应用程序也可以根据用户期望的练习强度，由软件智能规划适配的练习内容，并实时反馈练习结果。该软件适用于需要背诵巩固笔记、练习个性化题库的用户。
本软件的目标用户是有个性化刷题、知识点背诵巩固需求，希望自主定制题库并获得适配性练习规划的学习人群。比如希望通过某个题库软件来巩固笔记内容的人群，或者希望整合不同平台题目进行统一练习的人群。

本软件的亮点有这些：

1、 提供了题库管理框架，每个人都能创建自己的题库或笔记库。

2、 智能推荐允许相同的题库对不同掌握程度的人提供不同的推荐策略，实现个性化推荐。

3、 管理灵活，暂时不想练习的题目可以禁用，而非删除；可以同时导出、禁用不同题库各自的题目，导入单题时可以一次导入到多个不同的题库。

4、 题型模板多样灵活，比如创新支持填图任务，当需要对某流程图进行背诵、或组织切片图需要背诵时，该模板可以让用户在图片指定位置填写内容；填空题模板支持一空多解、复选列表和正确答案乱序识别功能。

5、 笔记功能灵活，单个笔记可以跨题库关联多个题目。

6、 接入了AI大模型，现在你可以在软件内使用你配置好的AI大模型批阅问答题。这代表所有支持的题型（问答、选择、判断、填空、填图任务）都可以自动批阅，反馈及时。
